Pattern Name
-------------
Two Pointers

WHAT?
* Use two indices instead of one.

WHY?
* Reduce repeated work.
* Reduce extra memory.
* Sometimes improve O(n²) to O(n).

WHEN?
* ✔ Palindrome
* ✔ Reverse
* ✔ Sorted Array
* ✔ Remove Duplicate
* ✔ Move Zeroes

WHEN NOT?
* ✘ Trees
* ✘ Graphs
* ✘ DP

Types
-----
1. Opposite Direction
2. Same Direction
3. Fast & Slow
